bank_Doc pruebas module¶

Módulo de pruebas de sistema para la aplicación bancaria.

Este módulo documenta las pruebas de sistema para la aplicación bancaria,

detallando cada paso que debe seguir el usuario para llevar a cabo las pruebas.

Pruebas documentadas:

  1. Creación de cuenta exitosa

  2. Prueba de autenticación exitosa

  3. Prueba de consulta de saldo exitosa

  4. Prueba de depósito exitoso

  5. Prueba de retiro exitoso

  6. Prueba de transferencia exitosa

  7. Prueba de estado de cuenta

  8. Prueba de cierre de sesión exitoso

  9. Logueo con credenciales no registradas

  10. Intento de creación de una cuenta ya existente

  11. Retiro de una cuenta sin fondos

  12. Transferencia a una cuenta no existente

  13. Ingreso de valores negativos en depósitos

doc_pruebas.test_autenticacion_exitosa()¶

Prueba 2: Autenticación exitosa

Pasos:

  1. Levantar el servidor: Se abre la terminal y se ejecuta el comando “python .bank_server.py”.

  2. Levantar el cliente: Se abre una segunda terminal y se ejecuta el comando “python .bank_client_antiguo.py”.

  3. Si aún no tiene una cuenta, debe crear una cuenta a través de los pasos para la «Creación de cuenta exitosa».

  4. En “Ingrese su opción:” se ingresa el número 1 que corresponde a la opción del menú de “Ingresar”.

  5. Se ingresa un ID existente, de una cuenta que ya haya creado previamente.

  6. Se ingresa la contraseña correspondiente a su «ID de cuenta».

  7. Se confirma el inicio de sesión exitoso con el mensaje «Ingreso exitoso».

Resultado esperado:

Mensaje que indica «Ingreso exitoso».
