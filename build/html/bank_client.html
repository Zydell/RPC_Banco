bank_client module¶

class bank_client.BankClient(server_url)¶

Bases: object

Clase que representa un cliente del banco.

Atributos:

proxy (ServerProxy): Proxy para comunicarse con el servidor RPC. current_account (str): La cuenta actual autenticada. notification_thread (threading.Thread): Hilo para recibir notificaciones. stop_notification_thread (bool): Bandera para detener el hilo de notificaciones. lock (threading.Lock): Lock para sincronizar las solicitudes RPC.
